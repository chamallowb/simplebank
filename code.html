Tutoriel : Comment déposer de l'argent sur son compte

Étape 1 : Rendez-vous dans une station-service (ville ou campagne)

Étape 2 : Parlez au PNJ (E)

Étape 3 : Cliquez sur "Je souhaiterais acheter un code de paiement."

Étape 4 : Faites Shift+Echap pour accéder à la console et copier le code de paiement

Étape 5 : Allez dans le salon 🤖・ℂ𝕠𝕞𝕞𝕒𝕟𝕕𝕖𝕤, puis faites /depot suivi du code de paiement

Étape 6 : Et voilà, votre argent a été déposé dans votre balance !
